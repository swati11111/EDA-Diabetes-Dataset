# EDA-Diabetes-Dataset
Exploratory Data Analysis (EDA) on Diabetes Dataset
This project performs Exploratory Data Analysis (EDA) on a diabetes dataset to understand patterns, detect anomalies, and extract meaningful insights before applying machine learning models.

**Objetive **
Understand data distribution
Identify missing or zero values
Detect outliers
Analyze feature relationships
Visualize patterns in the dataset

**Dataset**
Source

The dataset is obtained from Kaggle:

https://www.kaggle.com/datasets/mathchi/diabetes-data-set

**Features**

Pregnancies
Glucose
Blood Pressure
Skin Thickness
Insulin
BMI
Diabetes Pedigree Function
Age
Outcome (0 = Non-diabetic, 1 = Diabetic)

**Tools & Libraries**
Python 
Pandas
NumPy
matplotlib
